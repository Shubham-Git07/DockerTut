# 📝 Spring Boot To-Do App

A simple **To-Do List application** built with **Spring Boot**, **Thymeleaf**, and **PostgreSQL**.  
This project demonstrates CRUD operations, database integration, and deployment to the cloud.

---

## 🚀 Features
- Add, update, delete tasks  
- Mark tasks as complete  
- PostgreSQL database integration  
- Responsive Thymeleaf UI  
- Deployable with **Docker** or to **Render Cloud**

---

## ⚙️ Tech Stack
- **Backend:** Spring Boot (Java)  
- **Frontend:** Thymeleaf, Bootstrap  
- **Database:** PostgreSQL  
- **Build Tool:** Maven  
- **Deployment:** Docker, Render  

---

## 🖥️ Run Locally
App runs at 👉 http://localhost:8080

🐳 Run with Docker
Build Docker Image
docker build -t todo-app .
